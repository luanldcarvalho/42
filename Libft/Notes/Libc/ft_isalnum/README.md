## ft_isalnum
|DESCRIPTION|
|---|
|Checks for an alphabetic character or a number. '0' to '9', 'A' to 'Z' and 'a' to 'z'|
|RETURN VALUE|
|---|
|The values returned are nonzero if the character c falls into the tested class, and zero if not.|
