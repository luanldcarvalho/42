## ft_strjoin
|Prototype|
|---|
|char *ft_strjoin(char const *s1, char const *s2);|

|Parameters|
|---|
|#1. The prefix string.|
|#2. The suffix string.|

|Return value|
|---|
|The new string. NULL if the allocation fails.|

|External functs|
|---|
|malloc|

|Description|
|---|
|Allocates (with malloc(3)) and returns a new string, which is the result of the concatenation of ’s1’ and ’s2’.|
